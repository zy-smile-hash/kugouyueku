🚀 酷狗乐库页面部署指南

获取互联网分享链接的完整教程

📦 方案一：GitHub Pages（推荐）

1

创建GitHub仓库

去GitHub创建仓库
2

填写仓库信息

  • 仓库名：kugou-music-lib（或其他你喜欢的名字）
  • 选择：Public（公开）
  • 不要勾选 Add README
  • 点击 Create repository
3

复制仓库地址并推送代码

创建完成后，复制仓库地址，然后在本地项目目录运行：

git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
4

启用GitHub Pages

  • 进入仓库的 Settings 页面
  • 左侧菜单找到 Pages
  • Source 选择：main 分支，/ (root) 目录
  • 点击 Save
5

获取分享链接

🎉 您的预览地址

https://你的用户名.github.io/仓库名/yuelu_interactive.html

💡 提示：GitHub Pages部署通常需要2-5分钟，请耐心等待！
