📤 推送代码到GitHub - 超详细教程

一步步教你如何推送代码

1 创建GitHub仓库（如果还没创建）

点击下面的按钮去创建仓库：

去GitHub创建仓库 →

填写信息：

  • Repository name: 输入仓库名（如 kugou-music-lib）
  • Public/Private: 选择 Public（公开）
  • 不要勾选 Add README, .gitignore, License
  • 点击 Create repository
2 复制仓库地址

创建成功后，你会看到这个页面，复制红色框中的地址：

📸 这里应该是GitHub仓库页面截图

你会看到类似这样的地址：

https://github.com/你的用户名/仓库名.git

💡 地址格式示例

如果你的用户名是 zhangsan，仓库名是 kugou-music-lib，

地址就是：https://github.com/zhangsan/kugou-music-lib.git

3 执行推送命令

回到终端，执行下面两条命令（记得替换成你的地址）：

git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main

🎯 完整示例

git remote add origin https://github.com/zhangsan/kugou-music-lib.git
git push -u origin main

💡 提示：

  • 如果提示输入用户名密码，输入你的GitHub账号
  • 如果开启了双重验证（2FA），需要用 Personal Access Token
4 验证推送成功

执行完命令后，刷新你的GitHub仓库页面，应该能看到文件了！

✅ 如果看到代码文件，说明推送成功！

5 启用GitHub Pages

推送成功后，按以下步骤开启在线预览：

  1. 在仓库页面点击 Settings（设置）
  2. 左侧菜单找到 Pages
  3. 在 Source 下拉选择：main 分支
  4. 目录选择：/ (root)
  5. 点击 Save
  6. 等待 2-5 分钟

🎉 完成！你的预览地址是：

https://你的用户名.github.io/仓库名/yuelu_interactive.html
