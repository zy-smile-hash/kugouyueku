1 什么是终端？

终端（也叫命令行、Terminal、CMD）就是一个用文字输入指令来操作电脑的窗口。

就像你平时用鼠标点击文件夹、打开文件一样，终端可以用文字命令来完成这些操作。

🖼️ 终端窗口大概长这样：

Last login: Wed May 27 10:30:00 on ttys001
user@macbook ~ % _

2 如何打开终端？

🍎 macOS
🪟 Windows
1

按键盘上的 ⌘ Command + 空格 键

会弹出一个搜索框

2

输入 Terminal 或 终端

3

按 回车 键

1

按键盘上的 Win 键（Windows徽标键）

2

输入 cmd 或 PowerShell

3

按 回车 键

3 如何进入项目目录？

📂 我们需要进入到 /workspace 文件夹

1

在终端中输入以下命令，然后按回车：

user@macbook ~ % cd /workspace
user@macbook /workspace %
2

验证一下是否在正确的目录，输入：

user@macbook /workspace % ls
README.md deploy.sh images
yuelu_image.png deploy-guide.html yuelu_interactive.html
push-guide.sh push-tutorial.html ...

如果看到这些文件名，说明位置正确！✅

4 推送代码到GitHub

📤 执行这两条命令

⚠️ 先确保：你已经在 GitHub 创建了仓库，并复制了仓库地址！

1

执行第一条命令（记得替换成你的仓库地址）：

user@macbook /workspace % git remote add origin https://github.com/你的用户名/仓库名.git

💡 把上面命令中的 你的用户名 和 仓库名 换成实际内容！

2

执行第二条命令：

user@macbook /workspace % git push -u origin main
Enumerating objects: 30, done.
Counting objects: 100% (30/30), done.
Writing objects: 100% (30/30), 4.5 MiB | 2.5 MiB/s, done.
To https://github.com/你的用户名/仓库名.git
* [new branch] main -> main

看到类似上面的输出，说明推送成功！🎉
